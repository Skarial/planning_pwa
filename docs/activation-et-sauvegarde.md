# Activation et sauvegarde de l’application

Ce document explique comment activer l’application, sauvegarder vos données et les restaurer en cas de changement de téléphone.

---

## Activation de l’application

Lors de la première utilisation, l’application nécessite une **activation par code**.

Cette activation permet :

- de contrôler l’accès au logiciel,
- de lier l’utilisation à un appareil,
- tout en fonctionnant **sans compte**, **sans serveur** et **sans connexion internet**.

### Première activation — étape par étape

1. Ouvrez l’application.
2. Un écran **« Activation requise »** s’affiche.
3. Un identifiant appelé **Device ID** est affiché à l’écran.
4. Transmettez ce Device ID à la personne ou au service en charge de l’activation.
5. Vous recevez un **code d’activation**.
6. Saisissez le code dans le champ prévu.
7. Appuyez sur **Valider**.

Une fois l’activation validée :

- l’accès à l’application est permanent,
- aucune nouvelle activation n’est demandée sur ce même appareil.

---

## En cas d’erreur d’activation

Si le message **« Code invalide »** s’affiche :

- Vérifiez que le code est saisi exactement (sans espace en trop).
- Vérifiez que le Device ID transmis correspond bien à celui affiché sur l’appareil.
- En cas de doute, redemandez un code.

---

## Sauvegarde des données (Export)

Il est fortement recommandé d’effectuer une sauvegarde de vos données dans les cas suivants :

- avant de changer de téléphone,
- avant une réinitialisation de l’appareil,
- à titre de sécurité.

### Que contient la sauvegarde

Le fichier de sauvegarde contient :

- votre planning,
- vos paramètres,
- votre activation.

Aucune donnée n’est envoyée sur internet.

---

## Restauration des données (Import)

La restauration permet de récupérer **toutes vos données**, y compris l’activation.

### Procédure de restauration

1. Installez l’application sur le nouvel appareil.
2. Importez le fichier de sauvegarde.
3. L’application redémarre automatiquement.

Si l’application était activée au moment de la sauvegarde :

- **aucun nouveau code d’activation n’est demandé**.

---

## Changement de téléphone

Pour changer de téléphone sans perdre vos données ni votre activation :

1. Sur l’ancien téléphone :
   - effectuez une **sauvegarde (export)**.
2. Sur le nouveau téléphone :
   - installez l’application,
   - importez le fichier de sauvegarde.

Le planning et l’activation sont restaurés à l’identique.

---

## Confidentialité et sécurité

- Toutes les données sont stockées **localement sur votre téléphone**.
- Aucune donnée n’est transmise à un serveur.
- Aucun compte utilisateur n’est requis.
- L’application fonctionne **hors ligne**.

---

## Résumé

- Activation requise une seule fois par appareil.
- Sauvegarde recommandée avant toute opération importante.
- Restauration simple et sans réactivation.
- Données privées, locales et maîtrisées.
